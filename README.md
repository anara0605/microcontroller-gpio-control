
# Embedded Systems GPIO Control Project

## Hardware
- Microcontroller: ARM Cortex-M4
- IDE: STM32CubeIDE
- Programming: C

## Setup
1. Clone repository
2. Open in STM32CubeIDE
3. Connect ST-Link programmer
4. Flash to target board

## Code Structure
- `main.c`: Contains GPIO configuration and main loop
- `led.c/h`: LED control functions
- System clock configured at 16MHz
